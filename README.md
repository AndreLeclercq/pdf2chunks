# pdf2chunks

Création de chunks vectorisé à destination d'une BDD vectoriel (Milvus) pour être exploité par des requêtes au sein d'un pipeline RAG.

## Installation

Pré-requis : `Python3`, `uv` 

```shell
uv venv
source .venv/bin/activate
uv sync
```

## Lancement

```shell
uv run start.py
```
